# 리눅스 시스템 교육과정

> 개발자 취업 준비생을 위한 리눅스 시스템 관리 실무 교육

---

## 📋 교육 개요

| 항목 | 내용 |
|:---:|:---|
| **훈련기간** | 50시간 (2주) |
| **대상** | 개발자 취업 준비 비전공자/전공자 |
| **교재** | 시스템 & 네트워크 우분투 리눅스 (한빛미디어) |

---

## 🎯 훈련 목표

- **라즈베리파이 데비안 환경**에서 ROS2를 운영하기 위한 능력
- **AWS EC2 서버**에 배포될 SpringBoot 서비스를 관리할 능력

---

## 📚 교재 구성

### 1부. 리눅스의 기본 (1~6장)
리눅스의 탄생과 발전과정, GNU와의 관계 등 기본적인 리눅스 개요를 살펴보고, 가상머신에 설치하는 과정을 단계별로 설명합니다.

- 파일과 디렉토리 관련 명령 사용법
- 파일 편집기 Vim 사용법
- Bash 셸 사용법
- 파일 접근 권한 설정
- 프로세스 관리 명령

### 2부. 리눅스 시스템 관리 (7~10장)
리눅스 시스템을 관리하는 데 필요한 기능과 명령을 배웁니다.

- 파일 시스템과 디스크 관리
- 시스템 부팅 및 종료
- 패키지 설치 및 소스 컴파일
- 사용자와 그룹 관리

### 3부. 리눅스 네트워크 서비스 관리 (11~14장)
리눅스에서 네트워크를 통해 서비스를 관리하는 명령을 다룹니다.

- 네트워크 설정
- 원격 접속 (Telnet, SSH, VNC)
- FTP 파일 송수신
- 웹 서버 및 데이터베이스 서버 설치/설정
- NFS와 Samba를 통한 디렉토리 공유

### 4부. 리눅스 보안 및 프로젝트 (15~16장)
리눅스 보안의 기초적인 내용을 다룹니다.

- 보안 기본 개념
- 시스템 로그 점검
- 방화벽 설정
- NMap, PAM 등 보안 도구 사용법
- 종합 프로젝트: 리눅스 시스템 구축

---

## 📌 지도 목표

리눅스 OS 관리를 위한 **파일 시스템의 이해**, **셸 사용법**, **프로세스 관리**, **네트워크 서비스 관리** 명령어를 습득하여 운영체제 시스템 관리 능력 배양
