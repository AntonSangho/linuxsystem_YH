# 훈련기간 : 50 시간 

# 대상 : 개발자로 취업을 준비하는 비전공자와 전공자 그룹  

# 교재 : 시스템 & 네트워크 우분투 리눅스, 한빛 미디어 

## 교재에서 다루는 내용 
1. 리눅스의 기본 (1~6장) 
- 리눅스의 탄생과 발전과정, GNU와의 관계등 기본적인 리눅스의 개요를 살펴보고, 실습을 위해 리눅스를 가상머신에 설치하는 과정을 단계별로 설명합니다. 그리고 리눅스를 사용하기 위한 기본 지식을 학습합니다. 리눅스의 기초 명령인 파일과 디렉토리 관련 명령의 사용법, 파일 편집기인 vim의 사용법을 배웁니다. 또한 명령을 해석하여 실행하는 기능을  수행하는 배시셀의 사용법을 살펴보고, 파일에 대한 접근 권한을 설정하는 방법과 프로세스를 다루는 명령을 익힙니다. 
2. 리눅스 시스템 관리(7~10장)
- 리눅스 시스템을 관리하는 데 필요한 기능과 명령을 배웁니다. 파일 시스템과 디스크 관리 명령, 리눅스 시스템을 부팅하고 종료하는 방법, 패키지를 설치하고 소스를 컴파일하는 방법을 다룹니다. 또한 리눅스 시스템에 사용자와 그룹을 생성, 수정, 삭제하는 명령을 익힙니다. 
3. 리눅스 네트워크 서비스 관리 (11~14장)
- 리눅스에서 네트워크를 통해 서비스를 관리하는 명령을 다룹니다. 네트워크 설정과 텔넷, ssh, vnc를 통한 원격접속, FTP를 통한 파일 송수신 방법, 웹 서버와 데이터베이스를 서버를 설치하고 설정하는 방법을 익힙니다. 그리고 NFS와 삼바를 통해 리눅스와 리눅스, 리눅스와 윈도 사이에서 디렉토리를 공유하는 방법을 살펴봅니다. 
4. 리눅스 보안 및 프로젝트(15~16장)
- 리눅스 보안의 기초적인 내용을 다룹니다. 보안에 대한 기본 개념을 이해하고 시스템로그를 점검하는 방법, 방화벽을 설정하는 방법을 살펴봅니다. 또한 NMap과 PAM 등 리눅스 보안도구의 기본적인 사용법을 익힙니다. 마지막 장에서는 지금까지 공부한 것을 종합적으로 활용하여, 리눅스 시스템을 구축하는 간단한 프로젝트를 경험해봅니다.  
# 지도 목표 : 리눅스 OS 관리를 위한 파일 시스템의 이해, 셀 사용법, 프로세스 관리 방법, 네트워크 서비스 관리 명령어를 습득하여 운영체제 시스템 관리 능력 배양
