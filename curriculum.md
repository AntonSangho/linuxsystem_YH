# 리눅스 시스템 수업 커리큘럼

## 과정 개요

| 항목 | 내용 |
|------|------|
| **훈련기간** | 50시간 (주 5일 × 5시간 × 2주) |

# 최종 프로젝트: Apache + MySQL 기반 웹 데이터베이스 시스템 구축

## 프로젝트 개요

교재 16장을 기반으로 Apache 웹 서버와 MySQL 데이터베이스를 연동하여 웹 기반 데이터 관리 시스템을 구축합니다.

## 구축 단계

### 1단계: 환경 준비
- 프로젝트용 사용자/그룹 생성
- 디렉토리 구조 설계 및 권한 설정
- 필요 패키지 목록 확인

### 2단계: Apache 웹 서버 구성
- Apache2 설치 및 기본 설정
- 가상 호스트(VirtualHost) 설정
- systemd 서비스 등록 및 자동 시작 설정

### 3단계: MySQL 데이터베이스 구성
- MySQL 서버 설치 및 보안 설정
- 프로젝트용 데이터베이스 생성
- 테이블 설계 및 생성
- 사용자 계정 생성 및 권한 부여

### 4단계: 웹 애플리케이션 연동
- PHP 설치 및 Apache 모듈 연동
- DB 연결 설정 파일 작성
- 데이터 CRUD 웹 페이지 구현
  - 데이터 조회 (SELECT)
  - 데이터 입력 (INSERT)
  - 데이터 수정 (UPDATE)
  - 데이터 삭제 (DELETE)

### 5단계: 보안 및 마무리
- 방화벽 설정 (80, 3306 포트)
- 서비스 상태 점검
- 로그 확인 및 문제 해결
- 문서화

## 평가 기준

| 항목 | 배점 | 세부 내용 |
|------|------|----------|
| Apache 웹 서버 | 20% | 정상 동작, 가상 호스트 설정 |
| MySQL 데이터베이스 | 25% | DB/테이블 생성, 사용자 권한 설정 |
| 웹 애플리케이션 | 30% | PHP 연동, CRUD 기능 동작 |
| 시스템 설정 | 15% | 사용자/권한/방화벽 설정 |
| 문서화 및 발표 | 10% | 구축 과정 정리, 발표 |

# AI 조수(ChatGPT) 활용 가이드

## AI 도구 활용 원칙

| 구분 | 내용 |
|------|------|
| **역할** | 검색 대체가 아닌 "문제 해결 파트너" |
| **준비** | ChatGPT 무료 계정 회원가입 |
| **핵심** | 효과적인 질문법 습득 |

## AI에게 질문하는 기본 템플릿

```
[상황]
Ubuntu 22.04에서 Apache 웹 서버를 설치하고 있습니다.

[수행한 명령어]
sudo systemctl start apache2

[오류 메시지]
Job for apache2.service failed because the control process exited with error code.
See "systemctl status apache2.service" and "journalctl -xe" for details.

[질문]
이 오류의 원인과 해결 방법을 알려주세요.
```

## 실습 중 AI 활용 규칙

| 구분 | 내용 |
|------|------|
| **먼저 시도** | 오류 메시지 직접 읽기 → man 페이지 확인 → 그래도 모르면 AI 질문 |
| **복사할 것** | 정확한 오류 메시지, 실행한 명령어, OS 버전 |
| **하지 말 것** | AI 답변 무조건 복붙 → 반드시 이해 후 적용 |
| **기록하기** | 해결된 문제와 방법을 개인 노트에 정리 |

## 상황별 AI 질문 예시

### 1. 명령어를 모를 때
```
Ubuntu에서 현재 실행 중인 프로세스 중 메모리를
가장 많이 사용하는 것 5개를 보는 명령어는?
```

### 2. 오류 해결
```
[오류 메시지 전체 복사]

위 오류가 발생했습니다.
Ubuntu 22.04, MySQL 8.0 환경입니다.
원인과 해결 방법을 단계별로 알려주세요.
```

### 3. 개념 이해
```
리눅스에서 chmod 755와 chmod 644의 차이점을
실제 사용 예시와 함께 설명해주세요.
```

### 4. 설정 파일 작성
```
Apache에서 /var/www/mysite를 DocumentRoot로 하는
가상 호스트 설정 파일 예시를 보여주세요.
```

## 주의사항 

1. **AI는 틀릴 수 있다** - 항상 검증 필요
2. **보안 정보 노출 금지** - 실제 IP, 비밀번호 등 입력하지 않기
3. **학습 목적 유지** - 답만 얻지 말고 "왜 그런지" 이해하기
4. **의존하지 않기** - 기본기는 반드시 직접 익히기
