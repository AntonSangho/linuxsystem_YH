## 1. AI 조수 활용법 소개 (30분)

### 왜 AI를 활용하는가?

- 검색 대체가 아닌 **문제 해결 파트너**
- 오류 메시지 해석 및 해결책 제안
- 명령어 옵션 빠른 확인

### ChatGPT 준비

- 무료 계정 회원가입: https://chat.openai.com
- 계정 생성 후 기본 사용법 안내

### AI 질문 템플릿

```
[상황]
환경 설명 (OS 버전, 작업 내용 등)

[수행한 명령어]
실행한 명령어

[오류 메시지]
전체 오류 메시지 복사

[질문]
원인과 해결 방법을 알려주세요.
```

### 주의사항

1. **AI는 틀릴 수 있다** - 항상 검증 필요
2. **보안 정보 노출 금지** - 실제 IP, 비밀번호 입력 금지
3. **답만 복붙하지 않기** - 이해 후 적용
4. **먼저 시도할 것**: 오류 메시지 읽기 → man 페이지 → AI 질문

## 2. 리눅스 소개 (1시간 30분)

### 리눅스란?

- 1991년 리누스 토르발즈가 개발한 오픈소스 운영체제 커널
- GNU 프로젝트 + Linux 커널 = GNU/Linux

### 핵심 개념

| 용어 | 설명 |
|------|------|
| 커널 (Kernel) | 하드웨어와 소프트웨어를 연결하는 핵심 |
| 셸 (Shell) | 사용자와 커널 사이의 인터페이스 |
| 배포판 (Distribution) | 커널 + 소프트웨어 패키지 조합 |

### 주요 배포판

| 배포판 | 특징 | 패키지 관리 |
|--------|------|-------------|
| Ubuntu | 데스크톱 친화적, 초보자 추천 | apt (deb) |
| Debian | 안정성 중시, 서버용 | apt (deb) |
| CentOS/Rocky | 기업 서버용, RHEL 호환 | yum/dnf (rpm) |
| Fedora | 최신 기술 테스트베드 | dnf (rpm) |

### 왜 리눅스를 배우는가?

- **서버 시장 점유율 90% 이상** (AWS, GCP, Azure)
- 라즈베리파이, IoT 디바이스의 표준 OS
- DevOps, 클라우드 엔지니어 필수 스킬

### 수업 목표와의 연결

- **라즈베리파이**: Debian 계열 (Raspberry Pi OS)
- **AWS EC2**: Ubuntu/Amazon Linux 사용

## 3. VMware 및 Ubuntu 설치 (2시간)

### 가상머신이란?

- 물리적 컴퓨터 안에 소프트웨어로 만든 가상 컴퓨터
- 실제 하드웨어 손상 없이 실습 가능
- 스냅샷으로 언제든 이전 상태로 복구

### VMware Workstation Pro 17 설치

1. VMware 공식 사이트에서 다운로드
2. 설치 진행 (기본 옵션 유지)
3. 라이선스 활성화

### Ubuntu 22.04 LTS 다운로드

- 공식 사이트: https://ubuntu.com/download/desktop
- ISO 파일 다운로드 (약 4.5GB)
- **LTS** = Long Term Support (5년간 지원)

### 가상머신 생성

| 설정 항목 | 권장값 |
|-----------|--------|
| 메모리 | 4GB (최소 2GB) |
| 디스크 | 40GB (최소 20GB) |
| CPU | 2코어 |
| 네트워크 | NAT |

### Ubuntu 설치 과정

1. 가상머신 시작 → ISO 부팅
2. "Install Ubuntu" 선택
3. 언어, 키보드 레이아웃 설정
4. 설치 유형: "Erase disk and install Ubuntu"
5. 사용자 계정 생성
6. 설치 완료 후 재부팅

### 설치 후 확인

```bash
# 버전 확인
lsb_release -a

# 커널 버전 확인
uname -r
```

## 4. 실습과제 1: 가상머신 환경 구성 및 스냅샷 생성 (1시간)

### 과제 목표

- VMware에서 Ubuntu 가상머신 생성
- 정상 동작 확인
- 스냅샷 생성하여 초기 상태 저장

### 수행 단계

1. **가상머신 생성**
   - VMware에서 새 가상머신 생성
   - Ubuntu 22.04 ISO로 설치

2. **기본 설정 확인**
   - 터미널 실행 (`Ctrl + Alt + T`)
   - `lsb_release -a` 명령 실행
   - `uname -r` 명령 실행

3. **스냅샷 생성**
   - VMware 메뉴: VM → Snapshot → Take Snapshot
   - 이름: "초기 설치 상태" 또는 "Clean Install"
   - 설명: 날짜 및 상태 기록

### 제출 내용

- 터미널에서 `lsb_release -a` 실행 화면 캡처
- 스냅샷 목록 화면 캡처

### 평가 기준

| 항목 | 배점 |
|------|------|
| Ubuntu 정상 설치 | 50% |
| 터미널 명령 실행 | 20% |
| 스냅샷 생성 | 30% |

### Q: 한글 입력이 안 돼요
