# 프로젝트 개요

리눅스 시스템 수업 커리큘럼 및 교육 자료 프로젝트

## 수업 정보

| 항목 | 내용 |
|------|------|
| 훈련기간 | 50시간 (주 5일 × 5시간 × 2주) |
| 대상 | 개발자 취업 준비 비전공자/전공자 (리눅스 완전 초보) |
| 교재 | 시스템 & 네트워크 우분투 리눅스 (한빛미디어, 1~16장) |
| 실습환경 | VMware Workstation Pro 17 |
| 평가방식 | 단원별 실습 과제 위주 |

## 훈련 목표

- 라즈베리파이 데비안 환경에서 ROS2 운영 능력
- AWS EC2 서버에서 SpringBoot 서비스 관리 능력

## 수업 구성

- **1주차 (Day 1~5)**: 리눅스 기초 및 시스템 관리
  - 환경 구축, 기본 명령어, Vim, 셸, 권한, 프로세스, 디스크, 패키지
- **2주차 (Day 6~10)**: 사용자 관리, 네트워크, 보안
  - 사용자/그룹, SSH, FTP, Apache, MySQL, 방화벽, 최종 프로젝트

## 최종 프로젝트

Apache + MySQL 기반 웹 데이터베이스 시스템 구축 (교재 16장)

## 특이사항

- ChatGPT 무료 계정을 AI 조수로 활용하여 오류 해결 학습
- 오류 메시지 복사 → AI 질문 템플릿 활용 → 이해 후 적용

- `README.md` - 원본 교육 요구사항
- `curriculum.md` - 전체 수업 커리큘럼 (Day 1~10 상세 계획)
