# 프로젝트 개요

리눅스 시스템 수업 커리큘럼 및 교육 자료 프로젝트

## 수업 정보

| 항목 | 내용 |
|------|------|
| 훈련기간 | 50시간 (주 5일 × 5시간 × 2주) |
| 대상 | 개발자 취업 준비 비전공자/전공자 (리눅스 완전 초보) |
| 교재 | 시스템 & 네트워크 우분투 리눅스 (한빛미디어, 1~16장) |
| 실습환경 | VMware Workstation Pro 17 |
| 평가방식 | 단원별 실습 과제 위주 |

## 훈련 목표

- 라즈베리파이 데비안 환경에서 ROS2 운영 능력
- AWS EC2 서버에서 SpringBoot 서비스 관리 능력

## 수업 구성

- **1주차 (Day 1~5)**: 리눅스 기초 및 시스템 관리
  - 환경 구축, 기본 명령어, Vim, 셸, 권한, 프로세스, 디스크, 패키지
- **2주차 (Day 6~10)**: 사용자 관리, 네트워크, 보안
  - 사용자/그룹, SSH, FTP, Apache, MySQL, 방화벽, 최종 프로젝트

## 최종 프로젝트

Apache + MySQL 기반 웹 데이터베이스 시스템 구축 (교재 16장)

## 특이사항

- ChatGPT 무료 계정을 AI 조수로 활용하여 오류 해결 학습
- 오류 메시지 복사 → AI 질문 템플릿 활용 → 이해 후 적용
- **실무에 필요한 내용 위주로 진행** (교재 내용 중 실무 활용도가 낮은 부분은 축소)

## 실습 과제 작성 가이드라인

### 난이도: 초급 (★★☆☆☆)

대상이 **리눅스 완전 초보**이므로 실습 과제는 다음 원칙을 따름:

| 항목 | 지침 |
|------|------|
| 복잡한 문법 | 함수, 반복문, 조건문 **사용하지 않음** |
| 스크립트 구조 | 단순 명령어 순차 나열 |
| 작성 방식 | 복사/붙여넣기로 완성 가능하도록 |
| 단계 구성 | 5단계 이내, 각 단계별 시간 명시 |
| 명령어 | 해당 Day에서 배운 명령어만 사용 |

### 실습 과제 구성 템플릿

```
Part 1: 실습 준비 (폴더/파일 생성)
Part 2: 명령어 연습 (개별 명령어 실행)
Part 3: 스크립트 작성 (복사/붙여넣기)
Part 4: 권한 설정 및 실행
Part 5: 결과 확인/저장
```

### 실습 환경 경로

```
/home/user1/projects/실습명/
```

예시:
- `/home/user1/projects/cron_lab/`
- `/home/user1/projects/linux_lab/`

### 제출 방식

- **Git으로 제출** (GitHub 등)
- 실습 완료 후 `git add`, `git commit`, `git push`

### 필수 포함 항목

- 예상 결과 출력 예시
- 자주 하는 실수 및 해결 방법
- 완료 확인 체크리스트
- **Git 제출 가이드** (add, commit, push 명령어)
